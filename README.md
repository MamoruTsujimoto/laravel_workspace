# Laravel Workspace

Steps
---------------------------------

### 1. Clone Workspace

Checkout laravel_workspace Repository:
```
$ git clone git@github.com:MamoruTsujimoto/laravel_workspace.git
```

### 2. Initialize

Move to directory:
```
$ cd laravel_workspace
```

Docker up:
```
$ docker-compose up -d
```

Access to php container:
```
$ docker-compose exec php bash
```

Create to Laravel Project:
```
$ laravel new
```

### 3. Access to site
```
http://localhost
```

Enjoy!
